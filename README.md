<!-- GETTING STARTED -->
## Project setup
* If you don't want to use vuenos(2 packages I write own), you can remove it from package.json before install node_modules

#### Prerequisites
```bash
npm install npm@latest -g
```

#### Installation
```bash
npm install
```

#### For development
```bash
npm run serve
```

#### For production
```bash
npm run build
```
